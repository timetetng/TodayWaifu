## 丨功能一览

- `今日老婆` / `娶婆娘`：抽鸣潮女角色，当天固定；可按概率抽群友
- `今日老公`：抽鸣潮男角色，需先在配置开启
- `今日萝莉`：从本地萝莉图库随机发一张，与老婆/老公/群友完全独立
- `娶群友`：只抽群友，不依赖鸣潮角色图片资源
- `娶群主`：已停用，相关代码已注释，不再注册命令，也不再缓存群主身份
- `抢老婆 @对方` / `送老婆 @对方`：抢走或送出当天抽到的鸣潮角色老婆
- `老婆更新记录`：实时获取 GitHub 更新记录，渲染成图片发送
- `老婆列表` / `老公列表`：查看本群当天记录
- `今日老婆总排行`：展示老婆被娶次数排行；配置图库账号后会同步云端总排行
- `今日老婆帮助`：发送图文帮助卡片，并注册到 GSCore「core帮助」一览页
- 主人命令：分配老婆给指定用户；创建 / 上传 / 查看 / 删除自定义老婆图片；下载 / 删除萝莉图库

## 丨命令

### 普通命令（所有人可用）

| 命令 | 说明 | 相关配置 |
| --- | --- | --- |
| `今日老婆` / `娶婆娘` | 抽鸣潮老婆，当天固定；可配置概率抽群友 | 今日老婆概率抽群友 |
| `今日老公` | 抽鸣潮老公（图库模式不可用） | 启用今日老公 |
| `今日萝莉` | 从萝莉图库随机发一张 | 需先由主人「下载萝莉图片」 |
| `娶群友` | 只抽群友 | 启用娶群友 |
| `抢老婆 @对方` / `抢今日老婆` / `抢婆娘` | 抢对方当天老婆，每天一次 | 启用抢老婆 |
| `送老婆 @对方` / `送今日老婆` | 把自己今天的老婆送给对方 | 启用送老婆 |
| `老婆更新记录` / `今日老婆更新记录` / `老婆更新日志` | 实时获取 GitHub 更新记录并渲染成图片 | 启用老婆更新记录 |
| `老婆列表` / `今日老婆列表` | 查看本群今日老婆记录 | 无 |
| `老公列表` / `今日老公列表` | 查看本群今日老公记录 | 无 |
| `今日老婆总排行` / `老婆总排行` | 展示老婆被娶次数排行；配置图库账号后同步云端总排行 | 启用云端总排行 |
| `今日老婆帮助` | 发送图文帮助卡片 | 无 |

### 主人命令（自定义老婆 / 萝莉图库）

| 命令 | 说明 |
| --- | --- |
| `分配老婆 @对方 角色名` | 直接把指定老婆分配给对方，覆盖对方当天老婆 |
| `老婆创建<角色名>` | 新建一个自定义老婆角色 |
| `老婆上传图片<角色名>`（附带图片） | 给该角色上传立绘，返回图片 ID |
| `老婆图片列表<角色名>` | 查看已上传图片及对应图片 ID |
| `老婆删除图片<角色名> <图片ID>` | 删除该角色的某一张图片 |
| `老婆删除<角色名>` / `老婆删除确认` / `老婆删除取消` | 删除整个自定义老婆（需二次确认） |
| `下载萝莉图片` / `删除萝莉图片` | 下载或清空萝莉图库，下载时自动查重 |

萝莉图库与「今日老婆 / 今日老公 / 群友 / 自定义老婆」完全无关，是独立功能，图包来源 `nnlmc/waifu-gallery`。

### Debug 模式

开启「Debug模式」后，主人可无限抽、指定角色名，且不写入当天列表：

```text
今日老婆 达妮娅
今日老公 忌炎
```

## 丨数据存储

插件运行时数据都写在 GSCore 的 `data/TodayWaifu/` 下（配置、每日记录、自定义老婆图片、萝莉图库等），**不写插件代码目录**，升级 / 卸载不会丢档。旧版本数据会在首次加载时自动迁移过来。

## 丨鸣潮角色图片

角色图片由控制台「图片数据源」决定：`local` 读取本地鸣潮角色资源（默认查找 `gsuid_core/data/XutheringWavesUID/`，可用「本地角色图片目录」自定义路径）；`gallery` 走远程图库接口，需要配置图库账号密码。两种模式下「今日老婆」「今日老公」都按各自角色 ID 对照表过滤候选，可在控制台分别指定自定义对照表路径。

注意：图库模式会从远程图库获取并发送图片，可能存在部分图片内容风险。是否启用请自行判断；因使用图库产生的任何风险由使用者自行承担，插件作者不承担责任。

`老婆更新记录` 需要 GSCore 自带的 HTML 渲染组件（或已安装 `pyrenderhtml`）。

## 丨常用配置

| 配置 | 默认值 | 说明 |
| --- | --- | --- |
| 图片数据源 | local | local 用本地图片目录，gallery 用远程图库接口；图库内容可能存在风险，启用前请自行判断 |
| 图库账号 / 图库密码 | 空 | 图片数据源为 gallery 时需要填写；使用图库产生的风险由使用者自行承担，插件作者不承担责任 |
| 启用今日老公 | 关闭 | 是否启用「今日老公」 |
| 今日老婆概率抽群友 | 关闭 | 「今日老婆」是否有概率抽到群友 |
| 启用娶群友 | 关闭 | 是否启用「娶群友」命令；「娶群主」已停用且配置项已注释 |
| 启用抢老婆 / 启用送老婆 | 开启 | 是否启用对应命令 |
| 启用老婆更新记录 | 开启 | 是否启用「老婆更新记录」 |
| 启用云端总排行 | 开启 | 图库模式下同步云端老婆总排行 |
| Debug模式 | 关闭 | 主人调试用，可无限抽取、指定角色 |

各类成功提示 / 抽群友等文字模板，以及角色图片目录、对照表路径等进阶配置，均可在控制台按中文配置名查看和自定义。「娶群主」相关配置已注释停用，控制台不再显示。

## 丨帮助图自定义

`今日老婆帮助` 发送的是内置图片 `help.png`，插件运行时**不渲染、不依赖浏览器**。改样式时编辑 `help_preview.html`，本地渲染截图后覆盖 `help.png` 再提交即可。

## 丨常见问题

**`今日老婆` 没图片**：先确认本地鸣潮角色资源目录存在；没有这些目录时鸣潮角色相关功能无法正常发图。

**`今日萝莉` 提示还没有图片**：先用主人账号发送 `下载萝莉图片`。

**`今日老公` / `娶群友` 没反应**：先在控制台开启对应配置。

**`娶群主` 没反应**：该功能已停用，命令注册、群主缓存监听、控制台配置都已注释。

**抢不了别人老婆**：常见原因是抢老婆功能关闭、对方今天没抽、自己已经抢过一次、抢的是自己，或对方抽到的是群友。
